# pixi-webpack-starting-template

<h1>Pixi application with webpack</h1>

<p>Simple pixi webpack application template for future projects.</p>
<p>It is a Template nothing more to make u a quickstart.</p>

<ol>

  <li>Enjoy</li>
</ol>
